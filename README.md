# Projet de diffusion

Petit projet pour expérimenter un processus de diffusion (noising/denoising) en PyTorch.

Contenu:
- `diffusion.py` : fonctions principales de noising et boucle d'entraînement minimaliste.
- `utils.py` : fonctions utilitaires pour manipuler des images et des données 2D.
- `toy_diffusion_2D.ipynb` : notebook de démonstration.
- `datasets/` : jeux de données utilisés.

Exécution rapide:

1. Installer les dépendances:

```bash
pip install -r requirements.txt
```

2. Utiliser le notebook `toy_diffusion_2D.ipynb` pour des exemples interactifs.
